# Done UI Prototype

This folder contains a standalone static single-page prototype used to iterate on UI philosophy before app implementation.

## Files
- `index.html`: page structure and content sections.
- `styles.css`: token-like variables and component/layout styles.
- `script.js`: minimal behavior for compact mode toggle.

## Purpose
- Use a moderate Canvas-inspired direction (not a visual clone).
- Validate clean, modern layout and hierarchy.
- Prepare for screenshot-based iteration.

## Run Locally
Open `index.html` directly in a browser.

## Iteration Workflow
1. Share screenshots or references.
2. Update top-level CSS variables in `:root` for quick visual shifts.
3. Adjust section blocks by IDs:
   - `#foundations`
   - `#workflow`
   - `#components`
   - `#philosophy`
4. Keep this prototype static until design direction is approved.

## Notes
- No backend integration.
- No production app routing.
- Intended as a design exploration artifact only.
